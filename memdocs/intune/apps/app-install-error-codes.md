# <a name="intune-app-installation-error-reference"></a>Informazioni di riferimento sugli errori di installazione delle app di Intune

Oltre a seguire le procedure indicate per risolvere i problemi relativi agli errori dell'applicazione, è anche possibile ottenere informazioni sugli errori specifici dell'app in base ai codici di errore restituiti. Dopo aver individuato un codice di errore, usare la descrizione e le informazioni aggiuntive per risolvere l'errore.

Questa sezione menziona sia la registrazione di tipo amministratore di dispositivi che la registrazione Samsung Knox. Per altre informazioni, vedere [Registrazione di tipo amministratore di dispositivi Android](../enrollment/android-enroll-device-administrator.md) e [Registrare automaticamente i dispositivi Android usando Knox Mobile Enrollment di Samsung](../enrollment/android-samsung-knox-mobile-enroll.md).
